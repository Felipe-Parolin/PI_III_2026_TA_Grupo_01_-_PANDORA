# Grupo 01 - PANDORA
# Sistema Inteligente de Prevenção e Detecção de Falhas em Maquinários via Análise de LLM

A presente proposta se insere no contexto da transformação digital e da crescente necessidade de gestão eficiente, preditiva e sustentável de infraestruturas e ativos físicos. 

Atualmente, organizações de diversos setores operam com equipamentos corporativos e maquinários que não possuem telemetria nativa e dependem de registros de manutenção fragmentados e desestruturados. 

Nossa proposta, intitulada PANDORA, consiste no desenvolvimento de uma plataforma web integrada a uma API em Python para a gestão, rastreabilidade e, principalmente, detecção e prevenção inteligente de falhas nesses equipamentos, utilizando Modelos de Linguagem de Grande Escala (LLMs). LLMs são sistemas de inteligência artificial avançados, treinados em vastos volumes de dados para compreender, gerar e analisar linguagem humana de forma contextual e preditiva.
